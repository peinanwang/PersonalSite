Employee Profile CRUD

Java EE; MySQL; Anugular

This is a JEE web application that I build to practice Java web developent and learn concepts of AngularJS. It is a simple web application that supports CRUD (Create, Read, Update, Delete) operations on employee profiles.

Technical stack and tools used for this project include: View (JSP, jQuery, Ajax, AngularJS 1), Model (JavaBean), Controller (Servlet), JDBC, MySQL, Maven.
